# UnityPractie

## Environment
        Unity2020.3.21.f1

## Bonus
### infinite ground spawner
### Music
### Jump Animation

## Rule
    初始頁面按Start進入遊戲，Exit退出遊戲
    遊戲中會生成樹木和柵欄障礙物，地形有左轉右轉和懸崖
    按鍵操作：W(跳躍)，A(左平移)，D(右平移)，Q(左轉)，E(右轉)
    掉到懸崖下即遊戲失敗
